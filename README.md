# ECM1414 Event Planner

## Running the Program

Open terminal:

python event_planner.py ../Input_Files/input_small.txt


## Requirements

Python 3.x only.

No external libraries required.


## Files

activity.py
- Activity class

algorithms.py
- Brute force and greedy algorithms

event_planner.py
- Main program


## Algorithm Choices

The program uses time as the primary constraint.

Brute force checks all possible subsets.

Greedy selects activities based on enjoyment/time ratio.
